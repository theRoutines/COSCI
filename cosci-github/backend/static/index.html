Available Endpoints

  • GET /characters - List all characters
  • POST /characters - Create a new character
  • GET /characters/{id} - Get a specific character
  • DELETE /characters/{id} - Delete a character
  • PUT /characters/{id}/rate - Rate a character

Getting Started

To create your first character, visit the Swagger UI and try the POST /characters endpoint.
